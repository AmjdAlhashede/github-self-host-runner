# Assignment: Computing Ethics Case Study

## Part 1: Assignment Instructions
**Total Marks: 20**

### Objective:
To help students understand ethical issues in computing and apply ethical thinking to real-life cases.

### Instructions:
* Choose **ONE** real-world case related to computing ethics.
* Examples of topics:
    * Privacy violation
    * Cybercrime or hacking
    * Intellectual property issues
    * Internet addiction
    * Data breach incidents
* Write a **short report (2-3 pages)** and answer the following:

### Required Sections:
1. **Case Description (5 marks)**
    * Explain the case clearly.
    * Include what happened, where, and when.
    * Identify the main issue.
2. **Ethical Issues (5 marks)**
    * What ethical problems are involved?
    * Who is affected? (users, companies, society).
3. **Analysis and Discussion (5 marks)**
    * Why is this issue important?
    * What are the consequences?
    * Do you think the actions were ethical? Why?
4. **Solution / Recommendation (3 marks)**
    * Suggest possible solutions.
    * How can this issue be prevented in the future?
5. **Presentation & Organization (2 marks)**
    * Clear writing and organization.
    * Proper use of English.
    * Neat formatting.
